lib_base

Информация

lib_base - набор базовых инструментов для разработки кроссплатформенных приложений.

Модули:

  • base/media Чтение и воспроизведение аудио и видео. Отрисовка кадров видео требует включения Qt. Включение: -DLIB_BASE_ENABLE_FFMPEG -DLIB_BASE_ENABLE_QT
  • base/threads Обёртка над WinApi потоками для Windows и над phthread для Unix. Независимый от платформы AtomicInteger
  • base/system Определение процессора, порядка байт, системы, поддержка ключевых слов C++
  • base/core Основные инструменты
  • base/qt Графический интерфейс на Qt (стили, виджеты и т.д.). Включение: -DLIB_BASE_ENABLE_QT
  • base/io Классы для работы с файловой системой
